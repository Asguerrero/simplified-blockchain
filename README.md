# Simplified Blockchain
A simple model of a blockchain that works as a cryptocurrency

## To be installed:
 Flask==0.12.2: pip install Flask==0.12.2
 requests==2.18.4: pip install requests==2.18.4

## Get and Post requests
1. /connect_node
2. /updated_chain
3. /add_transaction
4. /updated_mempool
5. /mine_block
